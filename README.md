# MOBOT Quercus

Research data, analysis scripts, and figures for the Missouri Botanical Garden *Quercus* project.

## Repository structure

- `scripts/` — R and shell analysis workflows
- `data/` — metadata, ADMIXTURE results, plastome files, outgroup data, NCBI metadata, and project documents
- `figures/` — generated maps, ADMIXTURE plots, library-preparation images, and manuscript figures

Run the top-level workflows from the repository root so their relative paths resolve correctly.

## Excluded local content

The repository intentionally excludes files larger than 20 MB at initialization and the local `Quercus arkansana Images/` collection. See `.gitignore` for the exact paths.
